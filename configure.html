Configure LiveControl Zoom

LiveControl Zoom is a desktop application. Configuration is done on your Windows machine inside the app, not on a web page.

Where to configure

  • Open the app on the computer where it is installed.
  • Use the main window to set Meeting ID, Passcode, Display Name, ZAK/join token (if needed), SDK JWT (for development), and automation settings (pre-launch, time window, database path).
  • Click Save to config to write settings to LivecontrolZoomAuto.config.json in the app folder.
  • To sign in or sign out with Zoom (OAuth), use Sign in with Zoom or Sign out in the OAuth section of the app.

After authorization

Once you have authorized the app with Zoom (Sign in with Zoom), the app stores tokens locally. You can change display name, database path, and timing at any time in the app and save to config. No separate “configuration page” is required.

For full setup and usage instructions, see our Zoom app documentation.
