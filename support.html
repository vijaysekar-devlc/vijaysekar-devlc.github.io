Common issues

  • “Sign in with Zoom” fails or redirect doesn’t work: Ensure the Zoom OAuth app has redirect URI http://localhost:38472/callback and that no other app is using that port.
  • Join fails (auth / token): Try signing out and signing in again with Zoom. For scheduled joins, ensure the Runner has run after you signed in so tokens are available.
  • Runner doesn’t launch the app: Check the database path and column names (EventDate, meeting ID, passcode) in config, and that events are within the time window and pre-launch settings.
